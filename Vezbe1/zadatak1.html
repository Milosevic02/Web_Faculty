Web citaci


Web citac je aplikacija koja je namenjena prikazu sadrzaja web stranica.
Stranice se mogu dobaviti kako iz fajl-sistema tako i putem TCP/IP mreze.
Protokol kojim komuniciraju web citaci i web server je HTTP protokol. Danas su
najpoznatiji sledeci web citaci:

  1. Microsoft Internet Explorer
  2. Mozila
  3. Opera

Obratite pažnju da su svi nazivi web čitača u gornjoj listi linkovi koji upućuju na
adrese navedene u zagradi!
